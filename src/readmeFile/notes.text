# 🔢 Counter App (React)

A simple and optimized counter app built using **React Hooks** like `useState`, `useCallback`, `useRef`, and component optimization with `React.memo`.

---

## 🚀 Features

- ✅ Increment and Decrement counter by custom value
- ✅ Track **previous** count using `useRef`
- ✅ Avoid **unnecessary re-renders** using `useCallback` & `React.memo`
- ✅ Fully responsive and styled with Tailwind CSS

---



## 🛠️ Tech Stack

- React (Functional Components)
- React Hooks: `useState`, `useRef`, `useEffect`, `useCallback`
- Tailwind CSS
- Component memoization using `React.memo`

---
